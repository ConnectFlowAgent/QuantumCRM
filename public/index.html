WhatsApp Cloud API — validación de conectividad

Compruebe el token y el identificador de número contra Graph API antes de persistirlos en el almacén de configuración.

Pendiente de verificación

Ubicación habitual: Meta for Developers → su aplicación → WhatsApp → API Setup.

Tras una verificación exitosa, los campos inferiores se completan; confirme y utilice Guardar configuración para registrar en base de datos.
